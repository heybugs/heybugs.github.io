# 自组织团队

敏捷宣言指出，最好的架构、需求和设计出自自组织团队

笔记关键词：自驱动

# 自组织系统特征：

  1. 分散控制，去中心化
  2. 不断适应环境改变
  3. 局部作用下自然浮现的结构
  4. 有效反馈（肯定和否定）
  5. 弹性（团队的自我修复和调整能力）

# 权力矩阵

二维：管理者的职责 和 团队自己的职责 所占的比例

从上到下

  1. 定调子 （整体方向的设定）
  2. 搭台子 （团队及组织环境的规划）
  3. 写谱子 （工作过程和进度的监控和管理）
  4. 唱曲子 （图案你对任务的执行）

从左到右

  1. 管理者领导型团队
  2. 自管理型团队
  3. 自规划型团队
  4. 自治理型团队

举例：

海尔转型：生态化、全员参与创业 、小微组织 生产什么产品去调研用户需求，用户说了算

华为：任总：让听得见炮火的人指挥

金胖子：笑呵呵 和你刚当上项目经理差不多 但是现实情况很多 比如

同仇敌忾 -> 同床异梦 -> 同室操戈

# 项目资源的特点：

  1. 临时性
  2. 流动性
  3. 松散性
  4. 复杂性

# 虚拟团队/分布式团队

虚拟团队：主要通过网络沟通

分布式团队：团队不同小组分布在不同地方
