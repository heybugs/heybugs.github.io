# 9.1 规划资源管理

# 9.1.1 项目人力资源管理与企业人力资源管理的区别

企业传统人力资源管理：选、用、育、留

  1. 选 选拔 社招、社招
  2. 用 安排合适的岗位 知人善用 把人放对地方
  3. 育 培养 才能发挥更大作用
  4. 留 留下（防止跳槽 人才流失） 持续作战

项目的临时性决定了人力资源管理也有临时性特点：借（请）、用、育、还

注意点：在规划项目人力资源时就要做好项目结束后人员遣散计划
